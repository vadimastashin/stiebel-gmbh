    Наличными при получении

    Оплата наличными при получении товара осуществляется наличными непосредственно курьеру или в точке самовывоза при получении.

    Банковской картой онлайн на сайте

    Оплата осуществляется через авторизационный (защищенный) сервер процессингового центра Банка Сбербанк с использованием Банковских карт следующих платежных систем:

    Для оплаты банковской картой необходимо положить товар в корзину нажав кнопку "в корзину" и в процессе оформления заказа нужно выбрать соответствующий способ оплаты ("Оплата банковской картой онлайн на сайте").

    В случае, если вы оплатили заказ банковской карточкой и затем отказались от него, возврат переведенных средств производится обратно на банковскую карту, с помощью которой был оплачен заказ.

    Банковской картой при получении

    Вы можете оплатить заказ банковской картой в точке самовывоза. Или курьеру с помощью мобильного POS-терминала (терминал для оплаты картами), с которым он прибудет к вам.

    Безналичная оплата по счету

    Безналичная оплата по счету для физических и юридических лицВозможна оплата по безналичному расчету для физических и юридических лиц.

    При оплате безналичным расчетом для предоставления счета необходимо отправить запрос с указанием данных о товаре (наименование, количество) и контактных данных по нашему электронному адресу. Отгрузка оборудования при этом производится после оплаты выставленного счета по предоставленным нами реквизитам в любом отделении банка РФ.

    Наложенный платеж (почта EMS)

    Оплата наложенным платежом EMSПри выборе данного способа оплаты, оплата производится непосредственно в момент получения товара в отделении почты EMS (Почта России).

  • Через сервисы электронных платежей

    С помощью электронного кошелька Яндекс.Деньги

    Оплата яндекс деньгиВы можете оплатить покупку непосредственно со своего кошелька Яндекс.Деньги. Оплата происходит мгновенно, никакие комиссии за транзакцию не взымаются. Личные данные, данные платежа и транзакций, передаются по закрытому протоколу и обрабатываются на защищенных серверах Яндекс. Оплата происходит в онлайн режиме.

    Наличными по коду через терминал

    Оплата наличными через терминалПри выборе данного способа оплаты, вы получаете индивидуальный код, с помощью которого можно внести наличные в салонах связи или в терминалах оплаты (банкоматы Сбербанка, «Евросеть», «Связной», терминалы МКБ и «Мобил Элемент») и оплатить Ваш заказ. Всего более 170 тысяч пунктов оплаты.

    С помощью электронных денег Webmoney

    Оплата электронными деньгами webmoneyПринимаются к оплате платежи из рублевых кошельков международной системы расчета Webmoney. Личные данные, данные платежа и транзакций, передаются по закрытому протоколу и обрабатываются на защищенных серверах Яндекс. Оплата происходит в онлайн режиме.

  • Безопасность данных

    Процесс передачи данных

    Для оплаты покупки Вы будете перенаправлены на платежный шлюз ОАО "Сбербанк России" для ввода реквизитов Вашей карты. Пожалуйста, приготовьте Вашу пластиковую карту заранее. Соединение с платежным шлюзом и передача информации осуществляется в защищенном режиме с использованием протокола шифрования SSL.

    В случае если Ваш банк поддерживает технологию безопасного проведения интернет-платежей Verified By Visa или MasterCard Secure Code для проведения платежа также может потребоваться ввод специального пароля. Способы и возможность получения паролей для совершения интернет-платежей Вы можете уточнить в банке, выпустившем карту.

    Настоящий сайт поддерживает 128-битное шифрование. Конфиденциальность сообщаемой персональной информации обеспечивается ОАО "Сбербанк России". Введенная информация не будет предоставлена третьим лицам за исключением случаев, предусмотренных законодательством РФ. Проведение платежей по банковским картам осуществляется в строгом соответствии с требованиями платежных систем Visa Int. и MasterCard Europe Sprl.

При любом способе оплаты вы получаете полный комплект документов предусмотренных законодательством РФ.
